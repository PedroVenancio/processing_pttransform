De ED50 para PT-TM06/ETRS89
================================

Descrição
-----------

Esta ferramenta usa como ficheiro de entrada um raster no "ED50 / Zona UTM 29N" (EPSG:23029). O ficheiro de saída será um GeoTIFF no Sistema de Referência PT-TM06/ETRS89 (EPSG:3763).


Parâmetros
----------

- ``Ficheiro de entrada [Raster]``: raster de entrada


Ficheiros de saida
-------

- ``Ficheiro de saída [Raster]``: raster de saída em formato GeoTIFF
